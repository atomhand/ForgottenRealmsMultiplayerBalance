This is an unoffocial balance patch for the Forgotten Realms mod for Dominion 6.
It is not intended to achieve true balance - or anywhere near - just tone down anything that is outright ridiculous.

Balance Changes
    * Nerf excessively cheap Blood summoned demon mages
        * Increase price for Bind Kyton, Summon an Erinye Seductress, Summon a Contract Devil, Summon Infernal Barghest
        * Douse -1 to Kyton and Erinye Seductress
    * Remove Sanguine Dowsing Rod (blood deserves the hit)

Fixes
    * Fix sailing for Lady Luck (she needs to fit in her own boat)
    * Murabir General should not be naked and unarmed
